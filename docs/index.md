## :mag: Paso 1 — Investigación previa

!!! tip "Antes de la primera sesión"
    Investigad los siguientes conceptos. No hace falta código, solo entender las ideas.

### :computer: Modelo cliente-servidor

¿Qué ocurre exactamente cuando escribes una URL en el navegador y pulsas Enter? Identificad quién es el cliente y quién es el servidor en ese proceso.

### :satellite: Qué es una API

Buscad ejemplos reales: Netflix, Spotify, la app del tiempo de vuestro móvil. ¿Cómo obtienen sus datos? ¿Qué papel juega una API en esa comunicación?

### :signal_strength: HTTP — métodos y códigos de estado

- **Métodos:** `GET`, `POST`, `PUT`, `DELETE` — ¿para qué sirve cada uno?
- **Códigos de estado:** ¿qué significa cada rango? (2xx, 4xx, 5xx)

### :page_facing_up: JSON

Aprended la sintaxis básica. ¿En qué se diferencia un objeto JavaScript de JSON? Es el formato de comunicación que usaréis durante todo el curso.

### :card_index_dividers: Bases de datos relacionales

Sin entrar en SQL todavía: ¿qué es una tabla? ¿qué es una fila? ¿cómo se relacionan dos tablas entre sí?
